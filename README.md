# Bone-density
利用opencv kmeans 對骨密度填充物進行影像分割(Image Segmentation)，透過二值化、開運算、高斯模糊、輪廓尋找對影像後處理，結合GUI達到方便快速分割目標物與背景，並計算分割出來的骨密度空洞像素面積值。
以下為分割的結果，像素面值會因相機距離與相機畫素有所不同，僅供參考：
